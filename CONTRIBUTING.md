# Contributing to Greasy Fork Scripts

First off, thank you for considering contributing to this repository! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## 🏁 How Can I Contribute?

### Reporting Bugs
- Check the [Issues tab](https://github.com/JD-YH03D/release/issues) to see if the bug has already been reported.
- If not, open a new issue. Include a clear title, a detailed description of the problem, and steps to reproduce the bug.
- Mention your browser and script manager version.

### Suggesting Enhancements
- Open a new issue with the tag "enhancement".
- Explain why this feature would be useful and how it should work.

### Pull Requests
1. **Fork the Repo:** Create your own copy of the project.
2. **Create a Branch:** `git checkout -b feature/AmazingFeature`
3. **Keep it Clean:** Follow the existing coding style (Standard JS or Prettier).
4. **Document Changes:** Update the README or script headers if necessary.
5. **Commit Changes:** `git commit -m 'Add some AmazingFeature'`
6. **Push to Branch:** `git push origin feature/AmazingFeature`
7. **Open a PR:** Describe your changes in detail.

## 📜 Coding Guidelines
- **Metadata Block:** Ensure the UserScript metadata (name, version, match, etc.) is correctly updated.
- **Performance:** Avoid unnecessary heavy loops or global variable pollution.
- **Security:** Do not use `eval()` or inject insecure external resources.
- **Comments:** Comment your code to explain complex logic.
